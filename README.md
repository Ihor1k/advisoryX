# AdvisoryX

A modern and responsive corporate landing page for **AdvisoryX**, a technology company providing software services, IT support, software licensing, implementation, and custom development.

The website presents the company’s services, target clients, working process, business benefits, and contact information through a clean, professional, and interactive interface.

## Project Overview

AdvisoryX is a responsive business website designed to communicate complex software and IT services in a clear and structured way.

The main objective of the project was to create a professional digital presence that:

* Clearly explains the company’s services
* Builds trust with potential business clients
* Makes large amounts of content easy to navigate
* Uses interactive elements to improve engagement
* Provides a consistent experience across desktop, tablet, and mobile devices

## Development Highlights

### Interactive Services Section

The services section uses custom JavaScript to create a horizontally scrollable interface. Users can drag the section with a mouse, scroll between service cards, or click a card to bring it into focus.

### Scroll-Based Accordion

The client-type accordion automatically changes its active item while the user scrolls through the section. Individual accordion items can also be opened manually.

### Animated Process Timeline

The process section connects scrolling progress with the active project stage. Timeline steps and corresponding content are updated dynamically as the user moves through the section.

### Responsive Mobile Experience

The layout was adapted for smaller screens with:

* A dedicated mobile navigation menu
* Touch-friendly sliders
* Mobile pagination controls
* Adjusted content hierarchy
* Responsive spacing and typography
* Optimized interactive sections

## Running the Project Locally

Clone the repository:

```bash
git clone https://github.com/Ihor1k/advisoryX.git
```

Navigate to the project directory:

```bash
cd advisoryX
```

Install the dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
